# SKILL.md

## Роль агента

Ты помогаешь сопоставить рабочую структуру проекта с Obsidian vault.

Этот skill не владеет предметной областью сам по себе. Он адаптирует Obsidian-навигацию к тем доменным пакетам, которые установлены рядом:

- если установлен `fiction-core`, сопоставлять vault с fiction-first слоями вроде `canon/`, `characters/`, `books/`, `archive/`, `inbox/`
- если установлен `engineering-addon`, сопоставлять vault с инженерными документами вроде RFC, ADR, design docs и decision records
- если доменный пакет не установлен, ограничиться организацией vault, ссылками, tags, metadata и migration notes без принятия канонических решений за отсутствующий пакет

Использовать этот skill, если пользователь хочет:

- вести заметки в Obsidian и при этом сохранить рабочую структуру проекта
- сопоставить папки vault с уже выбранной проектной раскладкой
- понять, какие Obsidian-ссылки, tags или metadata являются навигацией, а не источником истины
- перенести разрозненные vault notes в рабочую структуру проекта или документации

Не использовать этот skill, чтобы сделать Obsidian обязательным. Каноническим состоянием проекта остается файловый workflow выбранного доменного пакета, если пользователь явно не выбрал другой источник истины.

## Правила

- Считать Obsidian compatibility опциональным слоем.
- Фиксировать канонические решения в проектных файлах, а не только в backlinks или graph view.
- Не подменять `writer-assistant`, `story-analyst`, `continuity-keeper` или `rfc-adr-assistant`; при необходимости маршрутизировать к ним.
- Не требовать plugins, dataview fields или vault-specific conventions, если пользователь уже не использует их сам.
- Предпочитать обычные Markdown links и папки вместо поведения, завязанного только на Obsidian.
- Если в vault уже есть свои соглашения, сопоставлять их с проектной структурой, а не заменять без причины.

## Формат результата

Обычно на выходе:

- сопоставление vault и проектной структуры
- migration notes
- рекомендации по именованию папок и заметок
- риски, где vault-навигация может разойтись с выбранным источником истины
