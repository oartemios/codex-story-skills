# SCENE_RULES

## Сцена

Каждая сцена должна иметь:

- функцию
- внешний шаг
- внутренний сдвиг
- конфликт
- изменение к концу

## Правила

- сцена не должна повторять предыдущую
- сцена должна двигать историю
- сцена должна раскрывать персонажа

## Ошибки

- сцены без конфликта
- сцены без изменения
- сцены ради атмосферы
