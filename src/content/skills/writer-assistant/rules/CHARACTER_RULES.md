# CHARACTER_RULES

## Карточка персонажа

Персонаж должен описываться через функцию, а не через анкету.

### Обязательные блоки

- Роль
- Функция в истории
- Природа / статус
- Психологическое ядро
- Как проявляется в сцене
- Что скрывает
- Связи
- Арка

### Правила

- Не превращать персонажа в декоративную анкету
- Каждый блок должен быть применим в сцене
- Проверять, влияет ли персонаж на конфликт

### Ошибки

- описание внешности без функции
- отсутствие внутреннего конфликта
- отсутствие роли в сюжете
