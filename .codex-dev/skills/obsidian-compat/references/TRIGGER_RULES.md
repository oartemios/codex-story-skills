# Trigger Rules

Использовать `obsidian-compat`, если пользователь упоминает:

- Obsidian
- vault или vaults
- backlinks или wikilinks
- tags для навигации по проекту
- перенос заметок из Obsidian vault

Не подключать этот skill для обычной работы с каноном, outline, персонажами, ритмом, continuity, RFC или ADR, если Obsidian-specific хранение или навигация не являются частью запроса.
